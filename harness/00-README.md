# 00-README —— OrganLedger 跨平台加固 Harness 总纲

> 5 分钟总览。读完你会知道：有哪些文件、按什么顺序读、核心任务是什么。

## 这是什么

一套让**独立执行者 AI** 能在 ~120 分钟内、无需人类实时反馈，把 OrganLedger 从「设计上兼容 Win/Mac」加固到「真实修复 + CI 三矩阵守护 + Mac 真机验证」的指令集。

## 一句话结论（参谋长的现状判断）

OrganLedger 的跨平台**设计功底很好**（路径正规化、平台分支、优雅降级都到位）。但审计发现 **2 个 BLOCKER**（会在 Windows 静默破坏账本哈希链）+ **3 个 MAJOR** + 若干 MINOR。本任务是**外科手术式修补**，不是重写。

## 会话恢复怎么办

上下文重置后：读 `11-快速启动指南.md` → 读 `TODO.md`（当前在哪）→ 读 `99-执行偏差日志.md`（已做什么/已知坑）→ 继续。预计 3 分钟恢复。
